# CodeAlpha_Hotel_Reservation_System
Hotel Reservation System – Java

Abstract

The Hotel Reservation System is a Java console application that allows users to view available rooms, book rooms, and view booking records. The project uses Object-Oriented Programming (OOP) concepts and MySQL database connectivity through JDBC to store reservation details.

Features

- Display available rooms
- Book a hotel room
- View all bookings
- Store booking details in MySQL database
- Menu-driven console interface

Technologies Used

- Java
- JDBC (Java Database Connectivity)
- MySQL
- Object-Oriented Programming (OOP)

Classes Used

Main.java

Handles the main menu and user interaction.

HotelManager.java

Manages room display, room booking, and viewing bookings.

Room.java

Stores room number, room type, and availability.

Reservation.java

Stores customer name, room number, and booking days.

DBConnection.java

Establishes the connection with the MySQL database.

Algorithm

1. Start the program.
2. Display available rooms.
3. Show the menu options.
4. Read the user's choice.
5. If Book Room is selected, enter customer details and save the booking.
6. If View Bookings is selected, display all booking records.
7. Repeat until Exit is selected.
8. End the program.

Output

The system successfully books hotel rooms and displays booking details stored in the database.

Conclusion

This project demonstrates the use of Java, JDBC, MySQL, and Object-Oriented Programming concepts to develop a simple and efficient Hotel Reservation System.
